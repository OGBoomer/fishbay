The philosopy around Myfishbay is to make data driven buying decisions. Most sellers do this with research and experience. Myfishbay allows you to shorten the experience side as you can take your research with you to the buying location.

This first step in this process is to create a research profile. A research profile consists of a Brand - Category pairing.

If you have already used the brand you can select it from the dropdown list. If it is a new brand just type it in the text box. You then select the category you want to match with that brand and click the button.

You can match a brand with as many categories as you like but each one is a seperate profile.

Note: For the search to work properly you must type the brand exactly as Ebay has it in thier system including upper/lower case letters.
(Adidas does not equal adidas)

On the profile list you have 3 options. Run searches, add models, or delete the profile.

Some clothing items will have certain models you will want to include in your search.

For examples if you are researching Levi's Jeans you may want to add models for 501's or 511's. These models will then show up on your search page.

Again it is important to note that your models must match what Ebay has for that model in the left hand navigation.

When you get to the search page it is self explanitory. You just select from the drop downs and check boxes the item specifics you wish to run agianst the search profile then click the search button.

You can enter any keywords seperated by spaces or commas that you want to include in the search.

Selecting an item here is the same as checking the left hand navigation on Ebay.

You can run as many different combinations of item specifics as you wish. Some item specifics allow you to select multiple choices, for example you can search L and XL on the same search.

Your searches will populate below with the item specifics, relavent data, and three buttons

First button is used to re-run the search with the same item specifics to get current numbers. A second button to delete the search. A final button that is a link to Ebay with the search parameters.

This allows you to verify that all of your item specifics are represented.

The brand page is what you want to use while in the field buying inventory

The initial page is a list of brands with two options. One to go to the details and one to delete the brand

The final brand detail page gives you the data of all your research organized by category to quickly determine if it is a profitable item or not. There is also a link to go directly to the create a search page for each category.
